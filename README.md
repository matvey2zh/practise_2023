# Автоматизация рабочего места диспетчера транспортных средств

Дипломный проект, посвящённый разработке информационной системы для автоматизации рабочего места диспетчера транспортных средств.  
Проект реализован с использованием **Spring Framework** и **Hibernate** и направлен на оптимизацию процессов управления логистическими операциями.

---

## Цель проекта

Целью дипломной работы является разработка программного обеспечения, которое позволяет:
- автоматизировать обработку логистических заявок;
- сократить время работы диспетчера;
- минимизировать ошибки ручного ввода данных;
- повысить эффективность управления перевозками и ресурсами.

---

## Функциональные возможности

- Управление заказами на перевозку
- Ведение базы клиентов и контрагентов
- Назначение транспортных средств и водителей
- Отслеживание статусов перевозок
- Хранение и обработка логистических данных
- Формирование отчетов
- Разграничение прав доступа пользователей

---

## Используемые технологии

- Java
- Spring Framework
  - Spring Boot
  - Spring MVC
- Hibernate
- REST API
- SQL (PostgreSQL / MySQL)

---

## Архитектура приложения

Приложение построено на основе многоуровневой архитектуры:

- Controller — обработка HTTP-запросов
- Service — бизнес-логика приложения
- Repository — работа с базой данных (Hibernate / JPA)
- Model (Entity) — доменные сущности
